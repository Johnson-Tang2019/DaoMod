# 道途劫灭（Dao Mod）

NeoForge 1.21.1 中式修仙模组，根据 `describe/设定集.md` 实现境界修炼、宗门、法宝、灵植矿材、四界探索与传奇挑战。

## 安装

- Minecraft Java 1.21.1
- NeoForge 21.1.x
- Java 21
- Curios API 9.5.1+1.21.1（必需）

将构建出的 JAR 与 Curios 一同放入客户端和服务端的 `mods` 目录。多人游戏要求两端模组版本一致。

## 完整流程

1. 合成道途指南与蒲团，在灵植、灵矿附近打坐积累修炼进度。
2. 经历炼气、筑基、金丹、元婴、化神、炼虚、合体、大乘、渡劫九大境界；每境含九个小境界。
3. 加入四宗之一，使用丹药、兵刃与法宝成长；突破大境界时承受天劫和因果心魔。
4. 探索尘寰界、灵虚界、帝落渊、永恒仙域。灵虚界包含中天神土、东海雷泽、西极金戈、南疆火域、北极冥海、归墟深海、虚空天外。
5. 收集 42 件法宝、22 种灵植、20 种灵矿。全部材料可通过世界生成、合成、遗迹或敌人掉落获得。
6. 依次完成五组传奇挑战，迎战 25 位设定人物对应的异兽化敌人并取得终局奖励。

## 主要内容

- 84 项设定集法宝与资源，另有九把背景兵刃、饰品、丹药、挑战符和指南。
- 22 种灵植与 20 种灵矿均可放置、采集并在世界中自然生成。
- 九处宗门及远古遗迹，四类宗门敌人，25 类传奇首领。
- 29 个敌对生物分别使用独立烘焙模型、GPT 贴图和程序化待机、行走、攻击动画，不使用玩家人物模型。
- 御剑飞行、蓄力剑气、法宝能力、Curios 饰品、丹毒、因果、天劫、死亡跌境与境界属性成长。
- 服务端权威修为数据；登录、重生、跨维度同步；多人首领按有效参战者共享奖励。
- 运行时贴图均由 GPT 在线图像模型生成，源图归档在 `art/gpt_sources/`。

## 构建与验证

```powershell
.\gradlew.bat build
.\gradlew.bat runGameTestServer
```

当前自动化覆盖修为进阶、存档与网络往返、敌对实体、境界血量、世界资源、宗门遗迹、设定群系、剑气多人归属等关键流程。
